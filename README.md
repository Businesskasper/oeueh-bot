# oeueh-bot

Öüh Bot

## Installation

1. clone the repository

```shell
$ git clone https://github.com/Businesskasper/oeueh-bot.git
```

2. install node modules

```shell
$ npm i
```

3. create .env and paste token

## Start locally

```shell
$ npm run dev
```

## Test

```shell
$ npm run test
```
